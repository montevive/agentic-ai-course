# The Model Context Protocol: A Standard Interface Between Agents and the World

- **Authors:** Kimura, S.; Anders, L.
- **Year:** 2025
- **Keywords:** MCP, protocol, tool integration, interoperability, standards

## Abstract

Every agent framework reinvents tool integration. The Model Context Protocol (MCP) proposes a client-server standard in which tools, resources, and prompts are exposed by servers and consumed by any compliant agent. We analyze the protocol design, including its stdio and HTTP transports, capability negotiation, and security model, and report on 14 production deployments connecting agents to filesystems, version control, and databases. Standardized integration reduced median integration time from 3.2 days per tool to 4 hours, at the cost of a thin performance overhead.
